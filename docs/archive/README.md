For the current state of the product, read:

- `../../ARCHITECTURE.md` — current system layout, kept up to date
- `../../DEPLOY.md` — current deploy + operations
- `../../README.md` — what Panko is, in one screen

Treat anything in this folder as historical context, not as a spec of how the app works today.
